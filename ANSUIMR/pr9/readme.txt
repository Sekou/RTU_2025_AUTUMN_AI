Расчет энергии автомобиля и поиск управления.

Лагранжиан L механической системы — это функция, представляющая собой разность её кинетической энергии T и потенциальной энергии U, 
выраженная через обобщённые координаты и скорости системы L=T-U. Уравнения движения системы выводятся из лагранжиана с помощью принципа наименьшего действия, который гласит, 
что система движется по траектории, минимизирующей интеграл от лагранжиана по времени. Этот подход позволяет упростить решение задач по сравнению с применением второго закона Ньютона
